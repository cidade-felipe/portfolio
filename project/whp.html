World Happiness Prediction

Projeto de análise e modelagem para entender fatores associados ao Índice de Felicidade e criar uma camada de predição, com dashboard interativo para exploração dos resultados.

1. Introdução

Este projeto investiga dados do World Happiness para entender quais variáveis se relacionam com o Índice de Felicidade e como essas relações podem ser transformadas em um modelo preditivo. Além da análise, o projeto entrega um dashboard para exploração visual (mapa e correlações) e uma estrutura de preparação de dados para treinar modelos de forma repetível.

2. Perguntas de análise

As perguntas que guiam o projeto são: quais fatores têm maior associação com o Índice de Felicidade, como o índice varia ao longo do tempo, como o indicador se distribui globalmente e se é possível prever o índice a partir de variáveis socioeconômicas e institucionais (por exemplo PIB per capita, expectativa de vida, apoio social, liberdade de escolhas e confiança no governo).

3. Fonte e preparação

A preparação dos dados organiza a base para análise e para treino de modelos. O foco é padronizar colunas, tratar nulos, garantir tipos corretos, evitar vazamento de informação e criar uma base final consistente para EDA e modelagem. O pipeline abaixo resume o fluxo de ponta a ponta.

4. EDA e insights

Na exploração, a ideia é visualizar a variação temporal do índice e entender relações entre variáveis explicativas. Os gráficos de dispersão com linha de tendência ajudam a enxergar associações fortes e fracas, e o mapa permite ler o cenário global rapidamente.

4.2 Correlações visuais

As correlações ajudam a entender o que “anda junto” com o índice. Em geral, indicadores como PIB per capita, expectativa de vida, apoio social e liberdade de escolhas tendem a acompanhar o aumento do índice, enquanto variáveis ligadas a confiança institucional podem apresentar padrões menos lineares dependendo do recorte.

5. Dashboard (Power BI)

O dashboard consolida a exploração em três páginas principais: visão geral, mapa e correlações. Ele permite filtrar o ano e comparar padrões sem precisar abrir notebook, o que deixa o projeto mais apresentável e fácil de navegar.

6. Conclusão

O projeto mostra como dados sociais e econômicos podem ser transformados em análise visual e em uma abordagem preditiva, conectando exploração, interpretação e produto final (dashboard). O resultado é uma visão clara do comportamento do Índice de Felicidade no tempo e no espaço, além de um caminho consistente para estimar o índice a partir de variáveis explicativas.

Na prática, o valor aqui é duplo: entendimento dos padrões e uma estrutura replicável de preparação e modelagem. Isso deixa o projeto pronto tanto para storytelling quanto para evolução técnica.

7. Próximos passos

Como evolução, dá para adicionar comparação entre modelos, análise de importância de variáveis, validação temporal (treinar em anos anteriores e testar em anos seguintes), e uma camada de explicabilidade (por exemplo SHAP) para tornar a predição mais interpretável. No dashboard, dá para incluir uma aba “Predição vs Real” e um ranking de países com maior erro, que costuma render bons insights.
