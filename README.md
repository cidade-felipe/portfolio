# Portfólio — Felipe Cidade Soares

Portfólio profissional focado em dados, BI e engenharia, com projetos completos que vão do dado bruto até insights e entregas finais. O site foi publicado no GitHub Pages e reúne destaques, projetos detalhados, perfil e contato.

## Acesso rápido

- Site: [https://cidade-felipe.github.io/portfolio/](https://cidade-felipe.github.io/portfolio/)
- GitHub: [https://github.com/cidade-felipe](https://github.com/cidade-felipe)
- LinkedIn: [https://www.linkedin.com/in/cidadefelipe/](https://www.linkedin.com/in/cidadefelipe/)


## O que você encontra aqui

- Página inicial com visão geral e projetos em destaque
- Página de projetos com cases e links para repositórios
- Página de perfil com trajetória e stack
- Página de contato para oportunidades

## Destaques técnicos

- Stack: `HTML`, `CSS`, `JavaScript`
- Foco: `Python`, `SQL`, `Power BI`, `ETL`, `ML`
- UI moderna com seções premium, CTAs e cards de projetos

## Estrutura do projeto

- `index.html` — Home
- `projects.html` — Projetos
- `profile.html` — Perfil
- `contact.html` — Contato
- `src/css/style.css` — Estilos globais
- `project/` — Páginas individuais de projetos

## Rodando localmente

Abra `index.html` em seu navegador ou use um servidor local simples:

```bash
python -m http.server 8000
```

Depois acesse:

```
http://localhost:8000
```
